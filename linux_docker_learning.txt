1. to get into some directory: example- cd ./proj then press tab and it automatically appears like cd ./projects
2. to remove a docker container: docker rm -f conatainer-name
3. to run a container with some .env file: docker run --name learn-postgres-container -d -p 5433:5432 --env-file .env learn-postgres
4. to remove images:docker rmi image-id
5. to create an image:docker build -t image_name .
6. to view container status : docker images
7. to create a docker compose if the image is already specified : docker compose up -d

- to delete the container and delete the volume : docker compose down -v
- to delete the container but keep the volume: docker compose down
